About DevFolio

Building the future of developer portfolios

Our Mission

DevFolio was created to solve a simple problem: developers spend too much time building portfolio websites instead of showcasing their actual work. We believe that your portfolio should be about your projects, skills, and achievements - not about wrestling with CSS and deployment.

Our platform provides everything developers need to create beautiful, functional portfolios in minutes, not weeks. Focus on what matters - your work - and let us handle the presentation.

Edinburgh Napier University

DevFolio is developed as part of the SET09103 - Advanced Web Technologies module at Edinburgh Napier University's School of Computing, Engineering & the Built Environment.

This project demonstrates advanced web development skills using Python Flask, modern frontend technologies, and industry best practices.

Technology Stack

Backend

Python Flask, SQLAlchemy, Flask-Login, Flask-WTF

Frontend

Bootstrap 5, JavaScript, Jinja2 Templates, Chart.js

Database

SQLite (Development), PostgreSQL (Production)
